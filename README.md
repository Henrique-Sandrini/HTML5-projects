# HTML5 & Full Stack Projects

This repository contains academic and personal projects developed during the **Systems Analysis and Development** program at **FATEC**, focusing on Web Development concepts using HTML, CSS, JavaScript, and Full Stack technologies.

Each project is organized in its own directory and includes a dedicated `README.md` with detailed documentation.

---

## Projects Overview

### Verum 1.6.1
Static web application focused on front-end structure, modular navigation, and multimedia integration.

- Technologies: HTML5, CSS3, JavaScript
- Type: Front-end / Static Web
- Documentation:  
  [`Verum Project README`](Verum_1.6.1/Verum_1.6.1/Verum/README.MD)

### ERP System
Full Stack ERP (Enterprise Resource Planning) system developed as an academic project.

- Technologies:
  - Frontend: React.js, Bootstrap, Axios, i18n
  - Backend: Node.js, Express.js, SQLite, JWT
- Features:
  - Authentication with JWT
  - Products, Clients, and Orders management
  - Internationalization (Portuguese, English, Spanish)
- Documentation:  
  [`ERP System README`](ERP/README.md)

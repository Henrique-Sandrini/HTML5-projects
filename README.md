# Verum 1.6.1

## Project Description

Verum is a static web application developed to demonstrate the structure and organization of a complete website. The project includes multiple modules such as a landing page, login system, chatbot, forum, image gallery, and help page.

It is intended for educational purposes, prototyping, and presentation of fundamental front-end web development concepts, including file organization, page navigation, and the use of multimedia resources.

## Features

- Main landing page with system navigation  
- Login and registration system (simulated, no backend)  
- HTML-based chatbot  
- Forum pages with images and videos  
- Image gallery  
- User help page  

**Note:** All features are demonstrative only. There is no database connection or data persistence.

## How to Run the Project

Although the project is static, it is recommended to run it on a local server to avoid issues with relative paths.

### Using XAMPP (Apache)

1. Extract the file `Verum_1.6.1.rar`
2. Copy the original **Verum** folder to the XAMPP `htdocs` directory:
C:\xampp\htdocs\Verum

4. Open the XAMPP Control Panel and start **Apache**
5. Open your browser and access:
http://localhost/Verum/PagInicial/paginicial.html

No additional configuration is required.
